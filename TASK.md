## 1. ПЕРЕДЕЛАТЬ СТАРТОВУЮ СТРАНИЦУ (OverviewPage)

### Сейчас
Таблица со списком установок. Скучно, нет визуала.

### Должно быть
Сетка карточек установок (grid 3–4 колонки, все установки без группировки).

### Карточка установки содержит:
```
┌──────────────────────────────────────────────────┐
│  🏭 Цеха №1,2 Пиролиз                    🟢/🟡/🔴 │
│                                                  │
│  ┌──────────┐  Производительность: 2 735 т/сут   │
│  │  Донат   │  Невязка: 3.27% (−90 т)            │
│  │ изм/согл │  Аномалий: 3                       │
│  │   /Δ%    │  Простои: 0 дн                     │
│  └──────────┘                                    │
│                                                  │
│  ▼ Развернуть аномалии                           │
└──────────────────────────────────────────────────┘
```

### Донат-чарт (бублик) на карточке:
- 3 сегмента: Измеренное | Согласованное | Отклонение
- В центре бублика: % отклонения (невязки)
- Размер: компактный ~120×120px
- Цвета: синий (изм), зелёный (согл), красный/жёлтый (отклонение)

### Показатели рядом с бубликом:
- **Производительность** = сумма согласованного входа (consumed.reconciled) за выбранный период, т
- **Невязка** = средний imbalance_rel.measured × 100, % + абсолютное значение в тоннах
- **Аномалий** = количество сработавших детекторов
- **Простои** = количество дней с downtime

### Статус-badge (правый верхний угол):
- 🟢 Норма — невязка < порога, нет critical аномалий
- 🟡 Внимание — есть warn аномалии
- 🔴 Критично — невязка > порога×2 или есть critical аномалии
- ⬜ Простой — установка не работала

### Аккордеон (раскрытие):
Клик по карточке → под ней разворачивается панель:
```
┌──────────────────────────────────────────────────┐
│ Аномалии установки «Цеха №1,2 Пиролиз»          │
│                                                  │
│ 🔴 01.01 Невязка МБ — дебаланс 3.27%             │
│ 🔴 01.01 Δ прибор/согл — Смолы С6-С8: 18.5%      │
│ 🟡 01.01 Δ прибор/согл — Метан: 9.2%              │
│ 🔴 01.01 Δ прибор/согл — Потери техн.: 47.4%      │
│                                                  │
│ [Подробная аналитика →]  ← ссылка на UnitDetail  │
└──────────────────────────────────────────────────┘
```
Каждая строка: severity-иконка + дата + метод + описание.
Внизу — кнопка перехода на полную детализацию (UnitDetailPage).

### Общие KPI сверху страницы (над сеткой карточек):
5 плашек:
- Суммарный вход (согл), т
- Суммарный выход (согл), т
- Средняя невязка, %
- Всего аномалий (число)
- Простои (установко-дни)
